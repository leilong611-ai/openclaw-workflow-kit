# obsidian-openclaw-sync

Sync helper for OpenClaw-related Obsidian configuration across devices.

This module is mainly for macOS users who keep Obsidian vaults in iCloud and want a cleaner way to coordinate local OpenClaw structures with those vaults.

## What it does

- Detects iCloud Obsidian vaults
- Finds OpenClaw-related structures inside vaults
- Helps create symlink-based sync flows
- Supports multi-agent workspace naming patterns

- `SKILL.md`
- `run.sh`
- `scripts/sync_helper.py`
