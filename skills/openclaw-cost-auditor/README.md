# openclaw-cost-auditor

Minimal cost auditing helpers for OpenClaw deployments.

This module is intentionally simple. It is meant to give operators a starting point for building cost awareness around token usage and rough spend estimation.

## What it does

- Scans log files
- Extracts token counts
- Produces a rough cost estimate

- `scripts/audit.py`
