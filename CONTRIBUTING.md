# Contributing

Thanks for your interest in `openclaw-workflow-kit`.

This repository is focused on practical OpenClaw workflows that are useful in real daily work.

## What kind of contributions are helpful

- Real-world use cases
- Documentation improvements
- Setup clarifications
- Packaging cleanup
- Bug reports with reproducible steps
- Ideas for practical workflow modules

## What to include in an issue

If you open an issue, please try to include:

- What you were trying to do
- Which module you were using
- Your environment
- The exact error or unexpected behavior
- What you expected instead

## Project direction

The project prioritizes:

- practical value
- maintainability
- clear documentation
- incremental public development

## Before submitting major changes

For bigger changes, please open an issue or discussion first so the direction can be aligned before implementation work starts.
