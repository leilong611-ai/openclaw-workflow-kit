# openclaw-workflow-kit

一个面向真实工作流的 OpenClaw 工具仓库，重点覆盖知识协同、可观察性和日常自动化。

这个仓库不是为了堆概念，也不是为了做一次性 Demo。

我更在意的是：

- 工作流能不能复用
- 文档是不是清楚
- 维护成本高不高
- 普通人能不能真正接进自己的日常流程

## 当前收录内容

### 1. `obsidian-openclaw-sync`

用于处理 Obsidian 与 OpenClaw 多设备协同，重点场景是 macOS + iCloud。

### 2. `openclaw-skill-observability`

用于查看 recent errors、失败会话和大致成本情况。

### 3. `openclaw-cost-auditor`

用于做基础 token / spend 审计，帮助建立成本意识。

## 这个仓库的定位

我的背景更偏运营管理、流程设计和团队管理，不是传统程序员出身。

所以这里的内容天然会更强调：

- 真实使用场景
- 长期维护
- 可交付
- 可复制

而不是只追求“看起来很厉害”。

## 适合谁

- 正在真实使用 OpenClaw 的人
- 想把 AI Agent 接进知识库和工作流的人
- 不想只看炫技 Demo，而想看可落地方法的人
